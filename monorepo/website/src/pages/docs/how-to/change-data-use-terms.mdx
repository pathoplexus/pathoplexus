Sequences submitted with Restricted-Use terms can be released as Open data, making them freely available for unrestricted use by the research community.

Read more about what [Open data](/about/terms-of-use/data-use-terms#41-open-data) and [Restricted Use data](/about/terms-of-use/data-use-terms#42-restricted-use-data) mean, and how they enable responsible data sharing while respecting research interests.

### How to change sequences from Restricted Use to Open

**Important:** Changing sequences from Restricted Use to Open is a **permanent change**. Once sequences are released as Open data, they cannot be moved back to Restricted Use.

1. Navigate to the pathogen you'd like to change the Data Use Terms for (e.g., mpox)
2. Click 'Submit' in the top menu, then click 'View' to see your released sequences
   - Alternatively, if logged in, you can navigate directly to your sequences using this link pattern: `https://pathoplexus.org/{pathogen}/submission/groupID/released?column_dataUseTerms=true`
3. If helpful, use the filters on the left-hand side to select the particular set of sequences you would like to change
4. Check the boxes of the sequences you'd like to change the Data Use Terms for
5. Click the 'Edit data use terms' link towards the top right
   - If you want to modify all sequences on the page at once, you can just select 'Edit data use terms (all sequences)'
6. Select 'Open' as the new Data Use Terms
7. Confirm the change

The sequences will be released automatically — there is no need to release them in a separate step.
