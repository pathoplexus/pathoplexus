# Information for AI Agents

This document provides key information for AI agents working on the Pathoplexus codebase.

## Accession Format

Pathoplexus accessions use the prefix `PP_` followed by a unique identifier. 

**Correct format:** `PP_0049AAG` or `PP_0049AAG.1` (with version number)

**Incorrect format:** `PPX_0049AAG` (do not use `PPX_` prefix)

When providing examples or documentation about accessions, always use the `PP_` prefix, not `PPX_`.
